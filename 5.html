Лабораторная работа №5: Построение графиков GnuPlot.

Задачи

1. Установить gnuplot последней версии для своей операционной системы.

2. Найти данные для построения графика. Например у меня: население Москвы по данным переписи населения за каждый год.

Где первая размерность: год, вторая: число прописанных граждан РФ в городе.

3. Сохранить данные в текстовом файле. Каждая строка – новая точка на графике.

4. Средствами GnuPlot в командной строке нарисовать график своих данных

5. Попробовать поменять тип, форму, цвет линии (или точек) графика

6. Задать название ряда данных, установить подписи к осям

7. Подобрать функцию и аппроксимировать свои данные.

8. Линию аппроксимации нарисовать на том же графике, отобразить легенду (названия линий) на рисунке.

9. Попробовать поварьировать границы отображения данных (устанавливаются в квадратных скобках после команды plot).

10. Оформить генерацию графика в виде скрипта gnuplot (*.gp)

11. В скрипте задать любой понравившийся терминал (jpeg / png / eps / …) командой set terminal, и установить выходной файл командой set output.

12. Добиться правильной работы скрипта и корректного отображения итоговой картинки.

13. Добавить картинку в latex-документ из работы 4 и в тексте описать как получались данные.
