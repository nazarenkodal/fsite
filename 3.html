Лабораторная работа №3: Введение в TeX (тех)

Задачи

1. Установить компилятор TeX. Рекомендуемые сборки: для windows – MikTeX; для mac – MacTeX, для linux – texlive.

2. Выбрать и установить текстовый редактор для написания tex – файла. Рекомендуется среда разработки TeXstudio (кроссплатформенная), но подойдет любой текстовый редактор.

3. Создать .tex документ. В документе описать минимальную разметку, указать класс документа article.

4. Наполнить параграфы минимум тремя абзацами текста.

5. В текст добавить уравнение, после текста добавить рисунок.

6. Между параграфами добавить разрыв страницы.

7. В конце документа добавить автособираемое оглавление.

8. Добиться безошибочной компиляции документа компилятором pdflatex.

9. Загрузить pdf документ на сервер, создать на него ссылку на сайте.
