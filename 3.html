Курсовая Назаренко TeX

Лабораторная работа №3: Введение в TeX (тех)

Задачи Установить компилятор TeX. Рекомендуемые сборки: для windows – MikTeX; для mac – MacTeX, для linux – texlive. Выбрать и установить текстовый редактор для написания tex – файла. Рекомендуется среда разработки TeXstudio (кроссплатформенная), но подойдет любой текстовый редактор. Создать .tex документ. В документе описать минимальную разметку, указать класс документа article. В тексте документа указать свои фамилию, имя и отчество. Добиться безошибочной компиляции документа компилятором pdflatex (команда в консоли либо кнопка F5 в TexStudio). Подойдет и другой компилятор, напр. LuaLaTex, XeTeX, и др. После ФИО создать главу – название своей последней курсовой работы и любые два параграфа из той же работы. Наполнить параграфы минимум тремя абзацами текста. В текст добавить уравнение, после текста добавить рисунок. Между параграфами добавить разрыв страницы. В конце документа добавить автособираемое оглавление. Добиться безошибочной компиляции документа компилятором pdflatex. Загрузить pdf документ на сервер, создать на него ссылку на сайте.
